# Historial Fase 1 — índice compacto

Este archivo ya no contiene el historial completo. Para ahorrar tokens, leer:

```text
codex/pipeline/fase_1_sparse_global/historial/INDEX.md
```

Después abrir solo el historial de la subfase o tema necesario:

```text
codex/pipeline/fase_1_sparse_global/historial/por_subfase/
```

Regla nueva:

- no volver a acumular todo en este archivo;
- añadir entradas nuevas en `por_subfase/historial_<ID>.md`;
- actualizar `INDEX.md` con una línea de resumen y la conclusión;
- si una entrada no pertenece a una subfase, usar `historial_general.md`,
  `historial_infraestructura.md` o `historial_pruebas_tipicas.md`.
