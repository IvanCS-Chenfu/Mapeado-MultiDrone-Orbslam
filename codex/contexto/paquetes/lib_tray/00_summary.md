# 00_summary — lib_tray

Resumen: Librería C++ para generación de trayectorias (polinomio, velocidad trapezoidal, elipse). Consumida por `dron_individual`.

APIs / librerías: `gen_tray_pol3`, `gen_tray_veltrap`, `gen_tray_elipse`.

Test executables: `test_gen_tray_pol3`, `test_gen_tray_veltrap`.

Build: compilar `lib_tray` y `dron_individual` juntos si se trabaja en control.

Detalles en `generacion_trayectorias.md` y headers en `include/lib_tray`.
